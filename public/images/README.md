# Imágenes del Sistema Médico

## Imagen de Fondo del Login

**Archivo requerido:** `consultorio-background.jpg`

Este archivo se usa como imagen de fondo en la pantalla de login del sistema.

### Instrucciones:

1. **Nombre del archivo:** Debe llamarse exactamente `consultorio-background.jpg`
2. **Formato:** JPG o JPEG
3. **Dimensiones recomendadas:** 1920x1080 px (Full HD) o superior
4. **Tamaño máximo:** 2 MB
5. **Contenido:** Una foto del consultorio "El Chaparro de Guanta"

### Cómo agregar la foto:

1. Toma una fotografía del consultorio o usa una foto existente
2. Redimensión la imagen a las dimensiones recomendadas (opcional)
3. Optimiza el tamaño del archivo (opcional)
4. Renombra el archivo a `consultorio-background.jpg`
5. Colócalo en este directorio: `public/images/`

### Nota:

Si no colocas la imagen, el sistema mostrará un fondo gris sólido como alternativa.
