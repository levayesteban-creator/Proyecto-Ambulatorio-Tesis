Guía de Instalación

Sistema de Gestión de Salud - Consultorio "El Chaparro de Guanta"

Para alguien que nunca ha instalado un programa de desarrollo. Sigue cada paso en orden. No saltes ninguno.

IMPORTANTE: Antes de comenzar, verifica que tu computadora tenga Windows 10 o superior y conexión a internet.

Paso 1: Instalar Laragon

Laragon incluye PHP, MySQL y Apache

Estos son los programas que necesita el sistema para funcionar con la base de datos.

  1. Abrir el navegador (Chrome)
  2. Ir a: https://laragon.org/download
  3. Hacer clic en "Download" (la versión "Lite" es suficiente)
  4. Abrir el archivo descargado (laragon-*.exe)
  5. Si Windows pregunta "¿Quieres permitir que esta app haga cambios?", dar clic en Sí
  6. En el instalador:
    • Idioma: seleccionar Español
    • Clic en Siguiente
    • Carpeta de instalación: dejar como está (C:\laragon)
    • Clic en Instalar
    • Esperar a que termine
    • Clic en Finalizar
  7. Laragon se abre solo. Verás una ventana pequeña en la esquina inferior derecha

Paso 2: Instalar Git

Git sirve para descargar el programa desde internet
  1. Ir a: https://git-scm.com/downloads/win
  2. Hacer clic en "Download" (se descarga solo)
  3. Abrir el archivo descargado
  4. Si Windows pregunta, dar clic en Sí
  5. En el instalador, dar clic en Next en todas las pantallas (las opciones por defecto están bien)
  6. Clic en Install → Esperar → Finish

Paso 3: Instalar Node.js

Node.js es para compilar el frontend del programa
  1. Ir a: https://nodejs.org
  2. Hacer clic en el botón verde "LTS" (versión estable)
  3. Abrir el archivo descargado
  4. Si Windows pregunta, dar clic en Sí
  5. En el instalador, dar clic en Next en todas las pantallas
  6. Si pregunta instalar componentes adicionales, dar clic en Sí
  7. Esperar → Finish

Paso 4: Instalar Composer

Composer es para instalar las dependencias PHP
  1. Ir a: https://getcomposer.org/download/
  2. Hacer clic en "Composer-Setup.exe" (Windows Installer)
  3. Abrir el archivo descargado → Sí
  4. En el instalador:
    • Seleccionar "Install for all users" → Next
    • Marcar "Add to PATH" (debe estar marcado) → Next
    • En "PHP executable": buscar y seleccionar la ruta de PHP de Laragon. Generalmente es:
      C:\laragon\bin\php\php-8.x.x-Win32-vs16-x64\php.exe
    • Next → Install → Esperar → Finish

Paso 5: Instalar Visual Studio Code

Es el programa donde se escribe el código
  1. Ir a: https://code.visualstudio.com/download
  2. Hacer clic en "Windows" (el botón grande)
  3. Abrir el archivo descargado → Sí
  4. En el instalador: Aceptar términos → Next → Next → Install → Finish

Paso 6: REINICIAR LA COMPUTADORA

ESTO ES OBLIGATORIO. Sin reiniciar, los programas no funcionarán correctamente.
Guardar todo lo que tengas abierto → Inicio → Apagar o cerrar sesión → Reiniciar.
Después de reiniciar, abrir Laragon y verificar que Apache y MySQL estén en verde (activos).

Paso 7: Clonar el proyecto

Descargar el programa desde GitHub
  1. Hacer clic derecho en cualquier parte del escritorio
  2. Seleccionar "Abrir en Terminal" o "Open in Terminal"
  3. Escribir estos comandos uno por uno, presionando Enter después de cada uno:
cd Desktop
git clone https://github.com/levayesteban-creator/Proyecto-Ambulatorio-Tesis.git

Esperar a que descargue todos los archivos (puede tardar 1-2 minutos).

Paso 8: Ejecutar el instalador

Instalación automática del sistema
  1. En la misma terminal, escribir:
cd Proyecto-Ambulatorio-Tesis
.\instalar.bat
  1. El instalador te preguntará:
    • Email del admin: escribe tu correo (ejemplo: tu@gmail.com)
    • Contraseña del admin: escribe una contraseña segura
    • Nombre del admin: escribe tu nombre completo
  2. Esperar a que termine (puede tardar 3-5 minutos)
  3. Cuando diga "INSTALACION COMPLETADA", presionar cualquier tecla

Paso 9: Configurar el sitio web

Crear la dirección del sitio en Laragon
  1. Abrir Laragon (clic en el ícono en la esquina inferior derecha)
  2. Clic en Menu → Apache → vhosts
  3. Se abrirá un archivo de texto. Borrar todo y escribir:
<VirtualHost *:80>
  ServerName gestion-salud.test
  DocumentRoot "C:/laragon/www/Proyecto-Ambulatorio-Tesis/public"
  <Directory "C:/laragon/www/Proyecto-Ambulatorio-Tesis/public">
    AllowOverride All
    Require all granted
  </Directory>
</VirtualHost>
  1. Guardar el archivo (Ctrl+S) y cerrar
  2. En Laragon, clic en "Reload"

Paso 10: Acceder al programa

Abrir el sistema en el navegador
  1. Abrir el navegador (Chrome)
  2. En la barra de direcciones, escribir:
http://gestion-salud.test
  1. Verás la pantalla de inicio de sesión
  2. Escribir el email y contraseña que configuraste en el Paso 8
  3. Clic en Iniciar Sesión
  4. ¡Listo! El programa está funcionando
Resumen rápido (si ya tienes todo instalado):
git clone https://github.com/levayesteban-creator/Proyecto-Ambulatorio-Tesis.git
cd Proyecto-Ambulatorio-Tesis
.\instalar.bat

Solución de problemas

ProblemaSolución
"No se reconoce 'git' como comando"Reiniciar la computadora después de instalar Git
"No se reconoce 'node' como comando"Reiniciar la computadora después de instalar Node.js
"No se reconoce 'composer' como comando"Reiniciar y verificar que Composer se instaló con "Add to PATH"
No carga la páginaVerificar que Laragon esté abierto y que Apache y MySQL estén en verde
Error de base de datosVerificar que MySQL esté activo en Laragon (verde)
El frontend se ve sin estilosEjecutar en la terminal: npm run build
